# ADR-0012: Use a Local-First Control Plane With a Public Domain Protocol

## Status

Accepted — 2026-07-23

The product will run as a headless Rust control-plane service with a Tauri desktop client and first-class CLI, IPC, authenticated loopback HTTP, and JSON/NDJSON interfaces; MCP and future IDE integrations are adapters to the same versioned domain protocol. Core capture, lexical retrieval, and basic briefing remain local and operational without cloud inference. This shape provides an integrated application experience without replacing user-owned source tools or giving the desktop UI privileged behavior unavailable to agents and automation.
